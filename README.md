#### Para utilizar este proyecto, es necesario crear un archivo .env dentro de la carpeta api, y una en client a su vez establecer las siguientes variables:

#### Api:
```
MYSQL_HOST= host
MYSQL_USER= user
MYSQL_PWD= password
MYSQL_DATABASE= database
MYSQL_PORT= port
```
#### Client: 
```
REACT_APP_BASE_URL = localhost
```
